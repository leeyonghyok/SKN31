17-1. DOM 요소 선택

querySelector()와 querySelectorAll()로 요소를 선택하는 실습입니다. 결과는 화면과 콘솔(F12)에서 확인합니다.

17.1.1 querySelector() — 하나의 요소 선택

버튼을 클릭하면 태그, 아이디, 클래스 선택자로 요소를 하나씩 찾아 콘솔에 출력합니다.

연습용 제목

아이디 선택자로 찾을 문단입니다.

클래스가 item인 첫 번째 문단입니다.

클래스가 item인 두 번째 문단입니다.

클래스가 item인 세 번째 문단입니다.

17.1.2 querySelectorAll() — 모든 요소 선택

버튼을 클릭하면 클래스가 fruit인 모든 항목을 NodeList로 가져와 출력합니다.

  • 사과
  • 바나나
  • 포도

버튼을 클릭해 보세요.

17.1.3 NodeList와 배열의 차이

버튼을 클릭하면 NodeList가 배열이 아닌 것과, 전개 구문으로 배열로 바꾸는 과정을 콘솔에 출력합니다.
