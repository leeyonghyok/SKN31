15. 이벤트(Event) 처리

각 실습의 결과는 화면과 개발자 도구 콘솔(F12)에서 확인할 수 있습니다.

15.1 이벤트 등록

버튼을 클릭하면 addEventListener()로 등록한 함수가 실행됩니다.

아직 클릭하지 않았습니다.

15.2 이벤트 객체

버튼을 클릭하면 이벤트 객체의 type, target, currentTarget 정보를 출력합니다.

버튼을 클릭해 보세요.

15.3 입력 이벤트

입력창에 글자를 입력할 때마다 input 이벤트가 발생하여 현재 값을 보여 줍니다.

현재 입력값: (없음)

15.4 기본 동작 막기

폼을 제출해도 preventDefault() 때문에 페이지가 새로고침되지 않습니다.

아직 제출하지 않았습니다.

15.5 이벤트 전파(이벤트 위임)

목록의 항목을 클릭하면 부모 ul에 등록한 처리 함수 하나가 클릭된 항목을 구분합니다.

  • 홍길동
  • 이순신
  • 강감찬

항목을 클릭해 보세요.
